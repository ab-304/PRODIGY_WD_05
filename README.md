# PRODIGY_WD_05

## Task Description
Build a Weather Web Application using HTML, CSS, and JavaScript that fetches real-time weather information from a weather API based on user input.

## Features
- Search weather by city name
- Real-time temperature display
- Weather condition description
- Humidity information
- Wind speed details
- Modern cloud-themed UI
- Responsive design

## Technologies Used
- HTML
- CSS
- JavaScript
- OpenWeatherMap API

## How It Works
- Users enter a city name
- JavaScript fetches live weather data using the OpenWeatherMap API
- Weather details are dynamically displayed on the webpage

## API Used
OpenWeatherMap API
